# sokol_bitmap_font

## dependencies

- sokol
- sokol_gp

## installation

copy `bitmap_font.h`, `bitmap_font.c`, `img.h`, and `img.c` into your project.

## running test

```bash
git clone --recursive https://github.com/vimichael/sokol_bitmap_font.git
cd sokol_bitmap_font
mkdir -p build
cd build
cmake ..
make skbmf_test
./skbmf_test
```
